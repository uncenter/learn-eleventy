## Development

1. Clone the repository:

   ```
   git clone https://github.com/uncenter/learn-eleventy.git
   ```
3. Navigate to the `.app` folder, and install dependencies with `pnpm`:

   ```bash
   cd .app
   pnpm install
   ```
5. In the .app folder, run the following command to start the development server:

   ```
   pnpm start
   ```
